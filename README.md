# Panic Detector AI

An AI-powered Flutter application designed to detect panic or distress sounds and assist in emergency situations.

## Features

- 🎤 Real-time microphone monitoring
- 🤖 AI-based panic sound detection
- 📱 Flutter mobile application
- ⚡ Fast inference using TensorFlow Lite
- 🚨 Emergency alert workflow

## Tech Stack

- Flutter
- Dart
- TensorFlow Lite
- Android Studio

## Project Structure

lib/
assets/
android/
ios/

## Installation

1. Clone the repository
2. Run `flutter pub get`
3. Connect an Android device or emulator
4. Run `flutter run`

## Future Improvements

- GPS location sharing
